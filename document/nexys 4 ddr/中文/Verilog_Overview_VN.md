C. VERILOG 中的基本对象（在模块开头声明）
1. 寄存器 - 存储分配之间的数据（抽象数据存储）
reg [15:0] pc, acc; /* 两个 16 位寄存器 - 位 15 最重要。 */
寄存器时钟； /* 一位寄存器 */
...
pc = 'h23fa; /* 给 pc[15:0] 赋值 */
时钟=~时钟； /* 对时钟进行补码 */
2.内存-寄存器的特殊情况=>一组寄存器：
reg [7:0] mem1 [0:255]; /* mem1是256字节的内存*/
...
mem1[0] = 'b01010101; /* 写入mem1的地址0 */
注：reg[7:0]r； /* 1个8位寄存器 */
reg r [7:0]; /* 8 个 1 位寄存器 */
3.Nets - 代表电线（一个驱动程序）和总线（多个驱动程序）。
导线w1、w2； /* 单线 */
线[15:0] w3； /* 16 条线一组 */
三[15:0] dbus； /* 三态总线；无冲突结果*/
三重奏[0:7]总线2； /* 三态总线； OR 逻辑解决冲突 */
w[0:7]总线1； /* 有线或总线 */
（还有魔杖和三角）
电源0接地； /* 由 0 电源驱动 */
（也供应1）
三向量[31:0] d1； /* 只能更新整个总线 */
三标量 [31:0] d2； /* 可以更新单独的行 */
4.有4个“逻辑值”（0、1、x、z）和8种可能的信号强度（我们
将使用可以分配给每个的默认强度值）
网/注册。
0,1 => 标准逻辑值
x => 未定义值
z => 浮动（三态）条件（无活动驱动器）
D. 预定义的原语逻辑门
1.类型：and、nand、or、nor、xor、xnor、not、buf（标准门）
bufif0、bufif1、notif0、notif1（三态驱动程序）
2. 标准门的连接在列表中指定，以开头
输出：and (out, in1, in2, ...);
3.三态驱动器的连接：bufif0（输出、输入、控制）；
bufif0/notif0 - 低电平有效控制输入（另外两个高电平有效）
4.实例化：列表类型、延迟以及一个或多个门和连接。

F. 指定程序语句的事件时间
1.事件可以在指定的延迟后发生：
#10 a = b; // 以10个时间单位读取b并更新a
a = #10 b; // 现在读取b并以10个时间单位更新a
2.对于顺序语句，延迟是相对于语句的执行时间而言的
先前的声明（即延迟添加）。并发语句的延迟
都是相对于语句集的开头。
从时间 T 开始：
开始更新于： 分叉更新于：
#10 a = 0; (T+10) #10 a = 0; (T+10)
#10 a = 1; (T+20) #10 b = 1; (T+10)
#10 a = 0; (T+30) #10c = 0; (T+10)
结束结束
3、与其他事件相关的时序控制：
@r a = b; // 当 r 改变时更新
@(posege r) a = b; // 在 r 的上升沿更新
@(负边r) a = b; // 在 r 的下降沿更新
@(posege r 或 posege s) a = b; // 更新任一位置。边缘
[也可以使用“和”]
G. 连续分配
1.连续赋值给网络，而不是寄存器，来模拟
由一个或多个门或驱动器驱动网。作业需要
每当任何驱动程序发生变化时。
电线a；
...
赋值 a = b + c； /* 更新 net a 以了解 b 或 c 的任何变化 */
2.连续赋值也可以在net声明中指定。
例如：在 f 或 g 发生变化后更新总线 e 10 个时间单位：
线[7:0] #10 e = f ^ g;

H. 其他公约
1. 注释包含在 /* ... */ 中
行尾的注释遵循 //...
2. 数字格式：无尺寸 有尺寸
十进制：659 5'd3（5 位）
十六进制：'h837ff 16'hz（16 位高阻）
二进制：'b010101 4'b0101（4 位）
'b0101_1100_0110（为了清晰起见，使用 _）
3. 标识符、字符串、实数、整数值遵循 C 语言语法。
同样，表达式（大部分）遵循 C 语法。
4. 表达式中使用的运算符：
算术加法：+ 乘法：*
子: - 引号: / 余数: %
逻辑连接词 - 和：&& 或：||不是：~
按位逻辑操作 - 求补：~ 和：& 或：|异或：^
移位操作：<< >>（例如 a = b << 2）
关系操作：<、>、<=、>=、==、!=、===、!==
\___\_ \____\_ x 和 z 值
|必须匹配。
如果不明确，结果为 x。
条件：表达式 ? true_action : false_action
（如果表达式非零则执行 true_action，否则执行 false_action）
连接：{...} 例如：{cy, acc} = a + b;
