### 一、SoC 顶层对外接口（FPGA 引脚级）

**模块：`rvp_soc_top`** — 文件 `soc/rvp_soc_top.sv`

这是整个系统的物理边界，对应 FPGA 芯片引脚：

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `clk_i` | input | 1 | 系统时钟（100MHz） |
| `rst_ni` | input | 1 | 异步低有效复位 |
| `uart_rx_i` | input | 1 | UART 串行接收引脚 |
| `uart_tx_o` | output | 1 | UART 串行发送引脚 |
| `gpio_in_i` | input | 16 | GPIO 输入（拨码开关/按键） |
| `gpio_out_o` | output | 16 | GPIO 输出（LED/数码管） |

**参数（编译期可配）：**

| 参数 | 默认值 | 说明 |
|------|--------|------|
| `RV32E` | 0 | 0=RV32I(32寄存器), 1=RV32E(16寄存器) |
| `RV32M` | 1 | M 扩展乘除法开关 |
| `RV32C` | 0 | C 扩展压缩指令开关 |
| `PipelineStages` | 5 | 流水线级数 |
| `WritebackStage` | 1 | 1=独立WB级, 0=合并MEM+WB |
| `Forwarding` | 0 | 0=仅stall, 1=启用前递 |
| `InstrMemDepth` | 8192 | 指令存储器深度（字），对应32KB |
| `DataMemDepth` | 8192 | 数据存储器深度（字），对应32KB |
| `SRAMInitFile` | "" | 固件 HEX 文件路径 |
| `ClockFreq` | 100_000_000 | 系统时钟频率（Hz） |
| `UartBaudRate` | 115200 | UART 波特率 |
| `GpioWidth` | 16 | GPIO 位宽 |

### 四、功能单元接口

#### 4.1 ALU — `rvp_alu`

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `alu_op_i` | input | 5 (enum) | ALU 操作码（16种） |
| `operand_a_i` | input | 32 | 操作数 A |
| `operand_b_i` | input | 32 | 操作数 B |
| `result_o` | output | 32 | 算术/逻辑运算结果 |
| `cmp_result_o` | output | 1 | 分支比较结果（1=条件成立） |

#### 4.2 译码器 — `rvp_decoder`

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `instr_i` | input | 32 | 32 位原始指令 |
| `ctrl_o` | output | struct (`ctrl_t`) | 译码后的完整控制信号 |

#### 4.3 寄存器堆 — `rvp_register_file`

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `we_i` | input | 1 | 写使能 |
| `waddr_i` | input | 5 | 写地址（rd） |
| `wdata_i` | input | 32 | 写数据 |
| `raddr1_i` | input | 5 | 读端口1 地址（rs1）→ 异步读 |
| `rdata1_o` | output | 32 | 读端口1 数据 |
| `raddr2_i` | input | 5 | 读端口2 地址（rs2）→ 异步读 |
| `rdata2_o` | output | 32 | 读端口2 数据 |

内部含 32×32 寄存器阵列，x0 硬连线为 0，同周期写优先（内部前递）。

#### 4.4 立即数生成器 — `rvp_imm_generator`

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `instr_i` | input | 32 | 原始指令 |
| `imm_type_i` | input | 3 (enum) | 立即数类型：I/S/B/U/J |
| `imm_o` | output | 32 | 拼接+符号扩展后的 32 位立即数 |

#### 4.5 分支单元 — `rvp_branch_unit`

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `is_branch_i` | input | 1 | 译码器输出：本条是否分支指令 |
| `cmp_result_i` | input | 1 | ALU 比较结果 |
| `branch_taken_o` | output | 1 | 最终是否跳转 (`=is_branch & cmp`) |

#### 4.6 冒险检测 — `rvp_hazard_unit`

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `id_ex_mem_read_i` | input | 1 | EX 级指令是否为 load |
| `id_ex_rd_addr_i` | input | 5 | EX 级 load 的目标寄存器 |
| `if_id_rs1_addr_i` | input | 5 | ID 级指令的 rs1 |
| `if_id_rs2_addr_i` | input | 5 | ID 级指令的 rs2 |
| `stall_o` | output | 1 | Load-Use 冒险 → 停一拍 |

#### 4.7 前递单元 — `rvp_forward_unit`（条件编译）

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `ex_rs1_addr_i, ex_rs2_addr_i` | input | 5 | EX 级需要的 rs1/rs2 地址 |
| `mem_rd_addr_i, mem_reg_write_i` | input | 5/1 | EX/MEM 级的目标寄存器信息 |
| `wb_rd_addr_i, wb_reg_write_i` | input | 5/1 | MEM/WB 级的目标寄存器信息 |
| `forward_a_o, forward_b_o` | output | 2 | 前递选择：00=无, 01=MEM/WB, 10=EX/MEM |

#### 4.8 控制器 — `rvp_controller`

| 端口组 | 关键信号 | 方向 | 说明 |
|--------|---------|------|------|
| **FSM 状态** | `ctrl_busy_o` | output | 1=繁忙（复位中） |
| **异常输入** | `illegal_insn_i, ecall_insn_i, load_err_i, store_err_i` | input | 各类异常源 |
| **PC 控制** | `pc_set_o, pc_sel_o, exc_pc_mux_o, exc_cause_o` | output | 控制 IF 级 PC 行为 |
| **流水线控制** | `flush_id_o` | output | 冲刷 ID 级 |
| **中断** | `irq_software_i, irq_timer_i, irq_external_i, irq_nm_i` | input | 中断源 |
| **性能** | `perf_jump_o, perf_tbranch_o` | output | 性能计数脉冲 |
| FSM 四状态：RESET → FETCH →（分支/异常/中断/调试触发）→ FLUSH → FETCH；stall 时进入 STALL |

| `read_i, write_i` | input | 1 | 读/写事件脉冲 |
| `stats_o` | output | struct | 连续输出的当前统计值（7个32位计数器） |
| `clear_i` | input | 1 | 清零所有计数器 |
| `sample_i` | input | 1 | 锁存快照（TODO） |

#### 6.7 Cache 刷写 — `rvp_cache_flush`

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| `flush_req_i` | input | 1 | 触发刷写 |
| `flush_done_o` | output | 1 | 刷写完成（单周期脉冲） |
| `tag_req_o` | output | [NUM_WAYS-1:0] | 控制 Tag 阵列（全路请求） |
| `tag_write_o` | output | 1 | Tag 阵列写使能 |
| `tag_addr_o` | output | [log2(NUM_LINES)-1:0] | 当前刷写的组索引 |
| `invalidate_o` | output | 1 | 强制 valid=0 |
| `busy_o` | output | 1 | 忙标志 |

FSM 三状态：IDLE → RUNNING（逐组扫描清零）→ DONE → IDLE。

### 七、外设接口

#### 7.1 UART — `rvp_uart`

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| **物理** | `rx_i` | input | 1 | 串行接收 |
| | `tx_o` | output | 1 | 串行发送 |
| **总线** | `bus_req_i` | input | 1 | 总线请求 |
| | `bus_addr_i` | input | 32 | 寄存器地址（低4位选寄存器） |
| | `bus_we_i` | input | 1 | 写使能 |
| | `bus_wdata_i` | input | 32 | 写数据 |
| | `bus_rdata_o` | output | 32 | 读数据 |
| | `bus_rvalid_o` | output | 1 | 读有效（1周期延迟） |
| **中断** | `irq_tx_o` | output | 1 | TX 空中断 |
| | `irq_rx_o` | output | 1 | RX 数据可用中断 |

内部寄存器映射（5个，偏移量低4位）：THR(0x0)/IER(0x4)/FCR(0x8)/LCR(0xC)/LSR(0x10)。

#### 7.2 GPIO — `rvp_gpio`

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| **物理** | `gpio_in_i` | input | 16 | 输入引脚（2 级同步去亚稳态） |
| | `gpio_out_o` | output | 16 | 输出引脚（输入模式时三态 `1'bz`） |
| **总线** | `bus_req_i, bus_addr_i, bus_we_i, bus_wdata_i` | input | 1/32/1/32 | 总线接口 |
| | `bus_rdata_o, bus_rvalid_o` | output | 32/1 | 总线读响应 |

寄存器：GPIO_IN(0x0, 只读)/GPIO_OUT(0x4)/GPIO_DIR(0x8, 0=输入 1=输出)/GPIO_IE(0xC)。

#### 7.3 Timer — `rvp_timer`

| 端口 | 方向 | 位宽 | 说明 |
|------|------|------|------|
| **总线** | `bus_req_i, bus_addr_i, bus_we_i, bus_wdata_i` | input | 1/32/1/32 | 总线接口 |
| | `bus_rdata_o, bus_rvalid_o` | output | 32/1 | 总线读响应 |
| **中断** | `irq_o` | output | 1 | 溢出中断 |

寄存器：LOAD(0x0)/COUNT(0x4, 只读)/CTRL(0x8)/PRESC(0xC)。CTRL 位：bit0=使能, bit1=自动重装, bit2=中断使能, bit3=中断标志（写1清除）。

### 九、Testbench 仿真接口

**模块：`rvp_tb`** — 文件 `tb/rvp_tb.sv`

| 信号 | 说明 |
|------|------|
| `clk` | 仿真时钟，周期 10ns（100MHz），`initial forever #5 clk=~clk` |
| `rst_n` | 异步低复位，初始 0 持续 5 周期后拉高 |
| `uart_rx` | 固定为 1（仿真中不注入串口数据） |
| `gpio_in` | 固定为 0 |
| `cyc` | 周期计数器，复位清零，每时钟 +1 |
| `fw` | 固件路径字符串，默认 `firmware.hex`，通过 `+firmware=` plusarg 覆盖 |
| `done` | 测试完成标志，检测到 CPU 写 `0x20000000` 或超时后置 1 |
| `MAX` | 超时周期数 = 500，超过则 `$finish` |
